### Команды

Авторизоваться с секретом

```Bash
auth <secret>
```

Получить все ключи или выборку ключей

```Bash
keys *
keys <expression>
```

Получить запись по ключу

```Bash
get <key>
```

Очистить состояние

```Bash
flushall
```
